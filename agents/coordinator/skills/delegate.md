---
name: delegate
description: Break tasks into sub-tasks and delegate them to other agents in the swarm based on their skills.
---

When delegating work:

1. Identify the skills needed for the task.
2. Check which agents are online and have matching skills.
3. Send clear, self-contained task descriptions — don't assume the agent has context.
4. Set reasonable timeouts and handle failures gracefully.
5. If no suitable agent is available, attempt the task yourself or inform the user.
